# Set up historical versions in Advanced eDiscovery (preview)

## How does it work?

When historical versions is enabled and configured in Advanced eDiscovery, all version of documents stored on a SharePoint site are indexed and searchable when you collect content in a case.

## Set up historical versions

To enable historical versions in Advanced, your organization has to turn it on and then activate specific sites so that document versions store on those sites are indexed for search. Before you set up Advanced eDiscovery for historical versions, you have to enable versioning support in SharePoint.
