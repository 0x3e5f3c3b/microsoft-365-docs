# Azure and Breach Notification Under the GDPR

Microsoft Azure takes its obligations under the General Data Protection Regulation (GDPR) seriously. For information about how Azure protects against a personal data breach and how we respond and notify you if a breach occurs, see [Azure Breach GDPR Documentation.pdf](https://servicetrust.microsoft.com/ViewPage/GDPRBreach?command=Download&downloadType=Document&downloadId=964c2966-7b52-499d-9cec-8659d1274200&docTab=bd9262f0-3dc1-11e8-87d1-758b9aa5dead_Breach).
