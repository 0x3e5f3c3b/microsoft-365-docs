> [!TIP]
> We recommend downloading the bundled ([mdatp-nokext.mobileconfig](https://raw.githubusercontent.com/microsoft/mdatp-xplat/master/macos/mobileconfig/combined/mdatp-nokext.mobileconfig)) file, rather than the [individual](https://github.com/microsoft/mdatp-xplat/tree/master/macos/mobileconfig/profiles) *.mobileconfig* files. The bundled file contains the files you need.
>
> accessibility.mobileconfig
> fulldisk.mobileconfig
> netfilter.mobileconfig
> systex.mobileconfig
>
> If any of these files are updated, you need to either download the updated bundle, or download the each updated file individually.
