[Security Copilot](/copilot/microsoft-365-copilot-overview.md) in Microsoft 365 Defender supports incident response teams in applying remediation actions while investigating attacks. Utilizing Security Copilotâ€™s AI-powered data processing, security teams can immediately create accurate incident reports within Microsoft 365 Defender.

Responding to incidents in the Microsoft 365 Defender often require general familiarity with attacks and how to use the portal to apply remediation actions to stop the attacks. Security Copilot draws upon the context of an incident and learning from prior investigations to generate appropriate remediation actions. Using the responses generated by Security Copilot allows incident response teams to quickly resolve incidents based on their assessment of each response provided.

This guide outlines how to access the guided response capability of Security Copilot in Microsoft 365 Defender, including information on providing feedback.
